## java 설치

개발환경: MacOS

sdkman 설치
```
curl -s "https://get.sdkman.io" | bash

```

```
source "$HOME/.sdkman/bin/sdkman-init.sh"
```

설치 확인
```
sdk version
```
https://arc.net/l/quote/zeuoebgo

java 설치
```
sdk list java
```

tab을 이용하여 버전 선택할 수 있음
```
sdk install java [tab]
```

디폴트 설정
```
sdk default java {identifier}
```
